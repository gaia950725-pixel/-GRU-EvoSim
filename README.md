# EvoSim Repository Guide

## Quick Start (AI Studio/Vite)
This repository can be run locally with Node.js.

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in `.env.local`
3. Run: `npm run dev`

## Canonical project files
- Runtime entry (app): `index.html`
- Canonical latest snapshot: `docs/releases/EvoSim_latest.html`
- Version archives: `docs/releases/archive/`
- Canonical changelog: `docs/releases/EvoSim_CHANGELOG.txt`

## Governance / Operations docs
- Master prompt (policy charter): `docs/governance/MASTER_PROMPT.md`
- GitHub workflow operations: `docs/ops/PROJECT_OPERATIONS.md`
- Output specification: `docs/spec/SPEC_OUTPUTS.md`
- AI handoff protocol: `docs/ai/AI_HANDOFF_PROTOCOL.md`
