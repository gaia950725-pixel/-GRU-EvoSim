# Snapshots for longterm projects

1회 프로젝트로 실행하기 어려운 복합프로젝트의 구현을 위해 현황조사를 저주기 메모하고 기획 아이디어를 누적 메모하기 위한 공간입니다.
장기 프로젝트 스냅샷은 아래 구조로 관리합니다.

- 장기 프로젝트(테마)별 폴더 분리: `docs/snapshots/longterm-projects/<longterm-project>/`
- 스냅샷 파일명은 도메인 기준으로 명시: 예) `BUILDER_BLOCK_SNAPSHOT.md`
- 기획/검증/점검 결과는 별도 감사 파일로 누적: 예) `*_AUDIT.md`
- 스냅샷, 기획, 검증/점검은 사용자 요청이 있을때만 저주기로 진행(이를 현재 코드 현상파악에 반영하지 말것)

현재 등록된 longterm project:

- `builder-block-refactor`
