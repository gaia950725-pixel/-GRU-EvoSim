단축키 가이드


    

  • Space: 일시정지/재개
    • 

  • [ / ]: 시뮬레이션 속도 감소/증가
    • 

  • 1: 시스템 패널(좌측 서랍) 토글
    • 

  • 2: 역사 기록 패널(우측 서랍) 토글
    • 

  • 3 / I: 인스펙터 패널 토글
    • 

  • C: 컨트롤바(하단) 확장/축소 토글
    • 

  • Ctrl (Hold): 리더/부리더 영향권 원 표시(신석기/리더십 ON)
    • 

  • ESC: 열려 있는 서랍(좌/우) 닫기
    • 

  • G: 선택 개체 그리드 오버레이 토글
    • 

  • N: 3×3 이웃셀 표시 토글(그리드 오버레이)
    • 

  • R: 시야 반경 커버 셀 표시 토글(그리드 오버레이)
    • 

  • P: 미니 프로파일러 토글
    • 

  • F: 팔로우 포커스(선택 개체를 화면 중심에 유지, 줌 유지 / 드래그하면 해제)
    • 

  • Shift (Hold) + Move: 농지 타일 미리보기(농지에서만 하이라이트)
    • 

  • Shift + Click: 농지 타일 정보 선택
    • 




                    텍스트 입력창에 포커스가 있으면 단축키는 동작하지 않습니다.
